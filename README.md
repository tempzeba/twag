### 台灣人文地理資料整合

##### 資料範疇
* 與 OpenStreetMap 授權可能衝突
* 歷史資料，可能現況已經改變
* 資料具有決策參考價值
* 資料具有歷史考證價值

##### 資料蒐集標準
* 以 GeoJSON 格式表示
* 具備資料來源
* 避免因果關係
* 避免評論
